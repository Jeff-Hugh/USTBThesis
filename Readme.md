#UStbThesis
------
北京科技大学硕士（博士）毕业设计论文Tex模板
##项目简要
北京科技大学官方只提供word版本模板，但是也已陈旧不堪。希望此项目能填补北科在Tex方面的空白，造福科大学子。也希望更多的贝壳可以加入进来，一起Coding，欢迎提供bug、issue和code.
##项目详情
请使用CTEX宏包进行编译。
本项目部分借鉴自BUAAThesis(https://github.com/BHOSC/BUAAthesis).
------
###至2016年3月23日，论文封面信息指导教师只有两个。
------
##文件简要说明
|功能|文件|
|:----|:----|
|主文件|main.tex|
|各章节文件|chapter.tex|
|论文相关信息（作者，指导教师，论文题目等）|ThesisInfo.tex|
|摘要|MyAbstract.tex|
|致谢|MyThanks.tex|
|论文前序|MyPreface.tex|
|个人简历|biogrephy.tex|
|独创性声明（此部分不必修改）|statement.tex|
|论文数据集|dataset.tex|
|引用文献信息|ref.bib|
|模板文件|USTBThesis.cls|

各文件保存后对主文件编译即可（注意：各文件需保持编码格式相同）。
